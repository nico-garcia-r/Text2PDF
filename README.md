# Text2PDF
Module to convert plain text to PDF.

## Commands
<ul class="commands_readme">
    <li>Open TXT</li>
    <li>Set Font</li>
    <li>Save PDF</li>
</ul>

**In the example robot, its converts a .txt file to PDF using fonts styles.
